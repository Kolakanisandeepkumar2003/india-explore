# 🌍 India Explore Tourism Website

A fully responsive tourism website built using **HTML5, CSS3, and Bootstrap**. This project showcases popular tourist destinations in India with interactive UI components and detailed views.

---

## 🌐 Live Demo

https://kolakanisandeepkumar2003.github.io/india-explore/

---

## 🚀 Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* 🏛️ Showcases famous tourist destinations like Taj Mahal, Golden Temple, Mysore Palace, and Varanasi
* 🎠 Bootstrap carousel for image galleries
* 🧭 Multiple sections with smooth navigation
* 🎨 Clean and user-friendly interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 4
